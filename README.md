## Broadband Business Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0206af6dd0f74202a9b9bc37762a6ee3/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Business Staff, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Announcements
6. Broadband
7. Personal Center
8. Repair Request
9. Broadband Payment
10. Broadband Favorites
11. Broadband Account Opening
12. Broadband Installation Appointment
13. Migration Application

**Business Staff Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Broadband Management
6. Broadband Message Management
7. Broadband Favorites Management
8. Broadband Account Opening Management
9. Repair Management
10. Repair Review Management
11. Repair Assignment Management
12. Forum Management
13. Announcement Management
14. Carousel Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Business Staff Management
5. User Management
6. Broadband Management
7. Broadband Message Management
8. Broadband Favorites Management
9. Broadband Account Opening Management
10. Broadband Payment Management
11. Broadband Installation Appointment Management
12. Repair Management
13. Repair Review Management
14. Repair Assignment Management
15. Forum Management
16. Announcement Management
17. Migration Application Management
18. Basic Data Management (Repair Type Management, Repair Status Management, Payment Type Management, Payment Status Management, Announcement Type Management, Broadband Type Management)
19. Carousel Management
